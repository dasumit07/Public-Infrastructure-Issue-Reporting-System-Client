# CityFix

A digital platform that allows citizens to report real-world public issues such as broken streetlights, potholes, water leakage, garbage overflow, damaged footpaths, and more. Government staff and admins can efficiently manage, verify, assign, and resolve reported issues.

## Features

1. Citizens can submit detailed reports including photos and location.  
2. Admin can review, verify, and assign issues to staff members.  
3. Staff can update the status of issues as Pending → In-Progress → Resolved → Closed.  
4. Citizens can track their issues in real-time and get updates on progress.  
5. Premium users receive priority support for faster issue resolution.  
6. Upvote system to help prioritize popular or critical issues.  
7. Payment system integrated with Stripe for premium subscriptions and issue boosting.  
8. Role-based access: Admin, Staff, and Citizen with specific permissions.  
9. Dashboard with analytics for Citizens, Staff, and Admins showing issue stats, payment stats, and recent activity.  
10. Search and filter issues by status, priority, category, or keywords.  
11. User management: Admin can block/unblock users and manage staff accounts.  
12. Timeline tracking for every issue to keep a clear history of updates.  
13. MongoDB used as database for scalable and efficient data management.  
14. Firebase authentication integrated for secure login and role verification.  

---

## How It Works

1. Citizens submit a report with issue details, photos, and location.  
2. Admin reviews the issue and assigns it to a staff member.  
3. Staff verifies the issue and updates its progress.  
4. System tracks the status from **Pending → In-Progress → Resolved → Closed**.  
5. Citizens receive notifications and can track their issue anytime.
